# 廃墟カフェ復興記 — プロット構成

## メインテーマ

**「場所は人が集まることで生きる」**

カフェという器は、コーヒーの味だけでは存在意義を持たない。
そこに通う人々が自分の物語を持ち込み、交わらせることで、
場所に「命」が宿る。

### サブテーマ

- **「選ばなかった道」との和解**: 時雨と雫の父娘関係、柊の退職、佐倉の閉店——
  全員が「もう一つの人生」を想像している。その選択を「失敗」ではなく
  「自分だけの道」として肯定する物語。

- **「続ける」ことの価値**: 大きな成功ではなく、日常を積み重ねること。
  カフェを毎日開け続けること自体が、小さな奇跡だと気づくまで。

## チャプター構成

### Ch.0「廃墟と最初の一杯」

**解放条件**: ゲーム開始時（プロローグ）

**プロットポイント**:
1. 柊が「月灯り」に初めて足を踏み入れる
2. 埃だらけの店内。唯一、古いエスプレッソマシンだけが丁寧に保管されている
3. 試しに一杯淹れてみる——苦い。でも、悪くない
4. 最初の客（佐倉）が来る。「……ここ、開いてるの？」

**引き**: 佐倉が去り際に「前の人は、もっと上手でしたよ」と呟く

**ストーリーキーワード**: はじまり、不安、最初の一歩

---

### Ch.1「最初の常連」

**解放条件**: 累計売上500達成

**プロットポイント**:
1. 佐倉が2度目に来店。柊が緊張するシーン
2. 天野（蓮）が初来店。商店街のバイト帰り
3. 宮内が初来店。「時雨さんの店を継いだのかい」——初めて前オーナーの名前が出る
4. 宮内から「月灯り」の簡単な歴史を聞く

**引き**: 宮内が帰り際に二階を見上げ、「あの部屋は……まだ開けない方がいいかもしれない」

**ストーリーキーワード**: 出会い、歴史の片鱗

---

### Ch.2「名前のないメニュー」

**解放条件**: レシピ5種発見 + 常連客2人

**プロットポイント**:
1. 柊が新レシピ開発に没頭し始める
2. 神崎（凛）が取材で来店。「廃墟カフェの復活」を記事にしたいと言う
3. 佐倉が「……その組み合わせ、こうした方がいい」と初めてアドバイスをくれる
4. 佐倉の「元パティシエ」という過去がほのめかされる

**引き**: 佐倉がうっかり「前の店では——」と言いかけて、黙る

**ストーリーキーワード**: 発見、距離が縮まる瞬間

---

### Ch.3「白い看板」

**解放条件**: 累計売上3000 + レシピ10種

**プロットポイント**:
1. 商店街に「Café BLANC」の出店告知が貼られる
2. 桐谷が「ご挨拶」として来店。柊のコーヒーを飲んで「美味しいけど、非効率ね」
3. 蓮が動揺。「また個人店が潰される」——実家のことと重なる
4. 宮内が「昔もこういうことがあった」と回想。商店街の歴史

**引き**: 桐谷が帰り際、メニュー表をじっと見て「……惜しいな」と小さく呟く

**ストーリーキーワード**: 危機、外部からの圧力

---

### Ch.4「自分のやり方」

**解放条件**: 特別接客10回 + Café BLANC開店イベント

**プロットポイント**:
1. Café BLANCが開店。洗練された内装、低価格、長い行列
2. 「月灯り」の客足が目に見えて減る
3. 柊が「チェーンのマネをすべきか」と揺れる
4. 佐倉が「あなたは、あなたの味を出せばいい」と初めて背中を押す
5. 蓮が「経営学的にはチェーンが正しい。でも——」と自分の矛盾に直面する

**引き**: 二階の段ボール箱から、前オーナーのレシピノートが見つかる

**ストーリーキーワード**: 葛藤、自分を信じる

---

### Ch.5「レシピノート」

**解放条件**: 隠しレシピ1種発見 + 親密度合計1000

**プロットポイント**:
1. レシピノートに時雨の手書きメモが残っている。レシピだけでなく、客の好みの記録
2. ノートの最後のページに「しずくへ——」で始まる書きかけの手紙
3. 宮内に聞く。「しずくさんって？」「……時雨さんの、娘さんだ」
4. 時雨に娘がいたこと、10年前に疎遠になったことが判明
5. 佐倉が自分の店を畳んだ経緯を語る——彼女のサイドエピソードが深まる

**引き**: ノートの手紙は途中で途切れている。「このカフェは——」の後が空白

**ストーリーキーワード**: 過去と現在、親子の距離

---

### Ch.6「それぞれの理由」

**解放条件**: 常連客3人の親密度★★★以上

**プロットポイント**:
1. 柊が「なぜ自分はこのカフェを続けているのか」と自問する
2. 常連客それぞれの事情が深堀りされる（各キャラエピソードが集束）
3. 神崎が取材記事を書き上げる。「読んでくれる？ その前に、一つ訊きたいことがある」
4. 「あなたにとって、この店は何？」——柊が初めて自分の言葉で答える

**引き**: 宮内が封筒を持ってくる。「時雨さんから預かったものがある。そろそろ、渡す時だ」

**ストーリーキーワード**: 覚悟、自分の言葉

---

### Ch.7「月灯りの手紙」

**解放条件**: メインストーリーキー5個 + 全常連客と特別イベント完了

**プロットポイント**:
1. 宮内から預かった封筒。中には時雨の手紙
2. 手紙の内容——カフェを閉めた理由、娘への想い、
   「この場所を次の誰かに」という願い
3. 柊が時雨の「不在」の意味を理解する
4. レシピノートの空白ページに、柊が自分のレシピを書き足す
5. 雫（時雨の娘）から電話が来る。「父のカフェが、まだ開いてると聞いて——」

**引き**: 「——行っても、いいですか」

**ストーリーキーワード**: 真相、継承、再会の予感

---

### Ch.8「ここにいる理由」

**解放条件**: 全メインストーリークリア条件達成

**プロットポイント**:
1. 雫が「月灯り」に来店。柊は知らない人にコーヒーを淹れる——ただの客として
2. 雫がカフェの変化に気づく。「……変わったんですね。でも、匂いは同じ」
3. 常連客が集まる日常のシーン。佐倉、蓮、神崎、宮内、桐谷
4. 雫が柊に言う。「ありがとうございます。父の場所を、あなたの場所にしてくれて」
5. エンディング——次の日の朝、柊がいつも通りカフェを開ける

**最後の一文**:
（今日も、いい香りだ）

**ストーリーキーワード**: 充足、日常の肯定、「続ける」ことの答え

## サイドクエスト設計方針

メインストーリーとは独立した短編エピソード。
常連客の親密度 or ストーリーキーで解放。

### サイドクエストの構造

1シナリオ = 3〜5シーン、テキスト量はメインの1/3程度。
メインテーマに共鳴するが、メインを読まなくても楽しめる。

### サイドクエスト例

| タイトル | 関連キャラ | 解放条件 | 内容 |
|---------|-----------|---------|------|
| 「忘れられたレシピ」 | 佐倉 | 親密度★★★ | 佐倉が昔の店のレシピを再現しようとする |
| 「父の背中」 | 蓮 | 親密度★★★ | 蓮が実家の金物屋跡地を訪れる |
| 「書かなかった記事」 | 神崎 | 親密度★★★ | 炎上した記事の真相と、書き直しの決意 |
| 「栞堂の秘密」 | 宮内 | Ch.5クリア | 古書店に眠る、時雨との交換日記 |
| 「白と月」 | 桐谷 | Ch.4クリア + 親密度★★ | 桐谷が本部に反旗を翻す日 |

## 展開テイスト指針

### やること
- 日常の小さな変化を丁寧に描く（季節、客の顔ぶれ、メニューの変化）
- 感情の爆発は抑制的に。「泣く」シーンは全編で1〜2回まで
- 「分かってもらえた」瞬間のカタルシスを大切にする
- 読後に「明日も頑張ろう」と思えるような余韻

### やらないこと
- 安易な恋愛展開（好意の表現はあっても、恋愛がメインにはならない）
- 悪役の設定（桐谷も含め、「敵」は存在しない）
- 過剰なドラマ（死、病気、事故など重い展開で感動を取りに行かない）
- ご都合主義的な解決（問題は丁寧に解きほぐす、魔法のように消えない）
