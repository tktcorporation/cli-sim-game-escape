# 廃墟カフェ復興記 — 文体ガイド

## ターゲットテイスト

**学園アイドルマスター × ラノベの心地よさ**

学マスのようにキャラクターの内面に深く寄り添いつつ、
ラノベの読みやすさとテンポ感を併せ持つ文体。

### 核心テクニック（リサーチから抽出）

- **育成メカニクスとシナリオの感情曲線を同期させる**: 経営が上手くいくとき＝シナリオでも前進。停滞期＝キャラの葛藤パート（学マス手法）
- **キャラの弱さの開示タイミングを設計する**: 第一印象→信頼→本音の3段階。開示は「設定」ではなく「物語」の中で行う
- **文体のギアチェンジ**: 日常パート＝口語的で軽い、シリアスパート＝体言止め・短文連打で緊張感（ラノベ手法）
- **1シーン1感情の原則**: 1つのシーンで複数の感情を詰め込まない。感情のピークは1つに絞る
- **章末に必ず「問い」を残す**: 情報の一部だけ開示し「なぜ？」を残す。読者に「続きを読みたい」と思わせる
- **「ハブ構造」の活用**: カフェという固定舞台に多様な人が来る構造。「注文」が会話の自然な起点になる（VA-11 Hall-A, ごちうさ手法）

### 参照作品（テイストの方向性）

| 作品 | 参照ポイント |
|------|------------|
| 学園アイドルマスター | キャラの内面描写の濃密さ、成長の描き方 |
| 氷菓（米澤穂信） | 日常の中にある小さな謎、抑制された文体 |
| ビブリア古書堂の事件手帖 | 「店」を舞台にした人間ドラマの空気感 |
| やはり俺の青春ラブコメは〜 | モノローグの味わい深さ、自意識の描写 |
| ゆるキャン△ | 日常の丁寧な描写、心地よい距離感 |

## 視点と語り

### 基本視点

- **三人称限定視点**: 主人公（柊）に寄り添う三人称
- 柊の内面（思考・感情）は地の文で直接描写する
- 他キャラの内面は「表情」「声色」「仕草」から推測させる
- 読者は柊と同じ情報量で物語を体験する

### 視点の例

```
【OK】
湯気の向こうで、佐倉が目を細めた。
何かを懐かしんでいるような——いや、ここにないものを
探しているような、そんな表情だった。
（この人、何を見ているんだろう）
柊はカップを置く音を、いつもより静かにした。

【NG】
佐倉は昔の店を思い出していた。あの頃の味と同じだと
感じて、嬉しくなった。
→ 佐倉の内面を直接書かない。柊から見える情報だけで伝える。
```

## 文体ルール

### やること

| ルール | 理由 |
|--------|------|
| 五感の描写を入れる | カフェは香り・音・温度の宝庫。読者を「そこにいる」感覚にする |
| 会話は短く、テンポよく | ラノベの読みやすさ。1セリフは原則2行以内 |
| モノローグは（）で囲む | 柊の内面を地の文と区別する。ラノベ的手法 |
| 段落は短く切る | ターミナルUIの画面幅を考慮。1段落3〜4行が上限 |
| 会話の後に一拍置く | 「沈黙」や「間」を地の文で描写し、テンポを作る |
| 比喩はカフェ・飲食に寄せる | 「苦いコーヒーのような沈黙」「砂糖が溶けるように」 |
| 章の最後は「引き」で終わる | 「続きを読みたい」で閉じる。クリフハンガー or 余韻 |

### やらないこと

| 禁止事項 | 理由 |
|---------|------|
| 説明的な地の文 | 「〜というのは〜だからだ」式の解説は避ける |
| 感情の直接記述 | 「嬉しかった」→「頬が緩んだ」に置き換える |
| 長い独白 | モノローグは3行以内。長くなるなら行動に変換する |
| 過剰な修飾 | 形容詞は1文に1つまで。名詞と動詞で勝負する |
| メタ的な言及 | 「ゲーム」「プレイヤー」「選択肢」等の言葉を使わない |
| 暴力・性的描写 | 全年齢対象。陰惨な過去があっても描写は抑制的に |
| 安易なオチ | 「実は幽霊でした」「夢でした」系の安易なツイストは禁止 |

## 会話文の書き方

### セリフの個性化ルール

各キャラのセリフは、名前を伏せても誰の発言か分かるように書く。

| 要素 | 説明 |
|------|------|
| 一人称 | キャラ固有（僕/私/あたし/俺/自分）。統一して崩さない |
| 語尾 | 特徴的だが自然な範囲。「〜ですわ」のような極端なものは避ける |
| 語彙レベル | 年齢・職業に合った言葉遣い。学生と社会人で差を出す |
| 話題の偏り | 好きなもの・気になることが会話に自然に現れる |
| 間の取り方 | 寡黙キャラは「……」が多い、饒舌キャラはセリフが連続する |

### 会話例（テイスト見本）

```
「——あの、コーヒー、もう一杯いいですか」
「もちろん。同じので？」
「……今日は、違うのを」
　佐倉がメニューに目を落とす。その指先が、ほうじ茶ラテの
行で止まった。
（ほうじ茶。この人にしては珍しい選択だ）
「ほうじ茶ラテですね。少しお待ちください」
「柊さんは、新しいもの試すのって好きですか」
　唐突だった。ほうじ茶を蒸らす手が、一瞬止まる。
「……嫌いじゃないですよ。少なくとも、このカフェを
始めたくらいには」
　佐倉が、初めて声を出して笑った。
```

## ノベルADV演出技法（ターミナルUI適用）

### 日常とシリアスの「ギアチェンジ」

CLANNADやFateが証明した原則: **日常の厚みがシリアスの重みを生む**。
日常を「退屈なパート」ではなく「後で壊されるものを丁寧に積み上げる」工程として設計する。

| フェーズ | 文体 | テンポ | ターミナル演出 |
|---------|------|--------|-------------|
| 日常パート | 口語的、軽い、ユーモアあり | セリフ多め、地の文は最小限 | 通常表示速度、BGM穏やか |
| 転換の瞬間 | 急に短文、体言止め | 一文ずつ表示、間が長い | 表示速度を落とす、一拍の空白行 |
| シリアスパート | 硬質、比喩的、五感描写が濃い | 地の文が増える、モノローグ重視 | 行間を広く取る |
| カタルシス | 感情の開放、短い一言 | 短文連打→沈黙→余韻 | 長い空白→決定的な一文 |

**切り替えのトリガー例**:
```
（日常）
「柊さん、この店にWi-Fiつけません？ 俺のレポートが——」
「ここは喫茶店だよ、蓮くん」
「喫茶店にWi-Fiがない方が珍しいっすよ今時——」

（転換）
　カランと、ドアベルが鳴った。
　見知らぬ女性が立っていた。
　スーツ。名刺入れ。そして、冷たい目。

「——ご挨拶に伺いました。Café BLANCの桐谷と申します」
```

### 麻枝准の「天丼」技法（フレーズの二重利用）

日常パートで繰り返されるフレーズやギャグを、シリアスパートで
再利用すると泣きのトリガーに変わる。（Real Sound 麻枝准インタビューより）

本作での適用例:
```
【日常（Ch.1-2）】
蓮「ココアおかわりいいすか」  ← 毎回言う。プレイヤーは「またか」と笑う

【シリアス（Ch.4）Café BLANC進出で客足激減】
カフェに誰も来ない夜。柊が一人でカウンターを拭いている。
ドアベルが鳴る。
蓮「——ココアおかわり、いいすか」
柊は声が出なかった。
```

他の天丼候補:
- 佐倉の「悪くないです」→ 最終話「……悪くない、どころじゃないですよ」
- 宮内の「いい香りだ」→ 最終話、柊のモノローグに移る
- 柊の「いらっしゃいませ」→ Ch.8で雫（時雨の娘）に言う時、声が震える

### 「泣きゲー」のカタルシス構造

Key作品（CLANNAD, リトルバスターズ）の「泣ける」構造を分析すると:

```
反復（日常の積み重ね）
  → 喪失/破壊（当たり前だった日常が崩れる）
    → 再構築（新しい形で日常を取り戻す）
      → カタルシス（「あの日常は無駄じゃなかった」と実感する瞬間）
```

本作での適用:
- **反復**: 毎日カフェを開け、同じ常連が来る日常のループ
- **喪失**: Café BLANCの進出で客足が減る/前オーナーの真相を知る衝撃
- **再構築**: 「自分のやり方」でカフェを続けると決める
- **カタルシス**: 雫（前オーナーの娘）が来店し「ありがとう」と言う瞬間
  — 日常の積み重ねが「誰かの救い」になっていたと知る

**重要**: 泣かせるために重い展開（死、病気）を使わない。
「日常が肯定される」瞬間のカタルシスこそ本作の武器。麻枝准の言う
「幸福の再発見」の構造。

### VA-11 Hall-A式「接客＝会話トリガー」

VA-11 Hall-Aの画期的発見: **ドリンクの選択が会話を分岐させる**。
正解のドリンクを出すと客が本音を語り、間違えると表面的な会話で終わる。

本作への適用:
- 特別接客で「好みに合うメニュー」を出すと **深い会話ルート** に入る
- 外したメニューを出すと **表面的な会話** で終わる（ペナルティではない）
- 「正解」は1つではない。ただし最も「刺さる」メニューが1つある
- これにより「この人の好みは何か？」がプレイヤーの知的好奇心を駆動する

**VA-11 Hall-A「注文聞き取り」の緊張感の応用**:
VA-11 Hall-Aでは「注文を聞き直してはいけない」というルールが
テキストを真剣に読ませる装置として機能した。

本作では、特別客の「本当に欲しいもの」が直接の注文ではなく
**会話の端々にヒントとして混ざる**設計にする:

```
佐倉「今日は少し冷えますね」         ← ヒント: 温かいものが欲しい
佐倉「……新しいメニュー、増えましたね」 ← ヒント: 新メニューに興味あり
佐倉「いつもので」                    ← 表面上の注文

→ 「いつもの深煎り」を出す: 安定ルート（+10）
→ 会話から「温かい新メニュー」を推理して出す: 深いルート（+25）
```

```
例: 佐倉への接客

[深煎りブレンド（いつもの）] → 安定した会話。親密度+10
[ほうじ茶ラテ（新メニュー）] → 「……新しいもの、試したんですね」深い会話。親密度+20
[カフェラテ（外し）]        → 「ありがとうございます」で終わり。親密度+5
```

### 「不在の存在感」の描き方

ノベルADVの名作に共通する技法: **登場しないキャラクターを最も印象的に描く**。
（Fateの衛宮切嗣、CLANNADの早苗の過去、シュタゲの「最初のDメール」）

時雨（前オーナー）の描き方:
- 直接登場しない。回想すら「他者の語り」越し
- **物を通じて語る**: レシピノート、エスプレッソマシン、ステンドグラス
- **空間を通じて語る**: 二階の段ボール、棚の配置、カウンターの傷
- **不在の重み**: 常連客が時雨の話をする時、いつも少し間が空く
- プレイヤーが「会いたかった」と思った時点で、設計は成功している

## シーンの構成法

### 基本構成（起承転結ではなく「入→展→転→余」）

| 要素 | 説明 | 例 |
|------|------|-----|
| **入** | シーンに入る。場所・時間・空気を1〜2文で | 「夕方の店内に、コーヒーの香りだけが残っていた」 |
| **展** | 会話や行動でシーンを展開する | 常連客との会話、新レシピの試作 |
| **転** | 小さな変化や発見。感情が動く瞬間 | 「佐倉の言葉に、柊は少し考え込んだ」 |
| **余** | 余韻を残して終わる。次への引きを含む | 「棚の奥に、見覚えのないレシピノートがあった」 |

### シーンの長さ目安

- メインストーリー: 1シーン = テキスト40〜80行
- キャラエピソード: 1シーン = テキスト20〜40行
- デイリー会話: 1〜3行のショートテキスト

## 感情表現の引き出し

シナリオ執筆時に使えるストック。直接的な感情語を避けるための変換表。

| 感情 | 身体表現 | カフェ的比喩 |
|------|---------|-------------|
| 嬉しい | 頬が緩む、声が弾む、手が踊る | 砂糖が溶けるように、温かい湯気のように |
| 悲しい | 視線が落ちる、声が小さくなる、手が止まる | 冷めたコーヒーのように、空のカップのように |
| 驚く | 目を見開く、手が止まる、息を呑む | カップを落としそうになる、湯気が揺れる |
| 怒る | 唇を引き結ぶ、声が低くなる、背筋が伸びる | 苦すぎるエスプレッソのように |
| 不安 | 指先でカップの縁をなぞる、目が泳ぐ | ミルクが濁っていくように |
| 安心 | 肩の力が抜ける、深く息を吐く | 最後の一口のような温かさ |
| 懐かしい | 目を閉じる、ゆっくり息を吸う | 記憶の中のあの一杯、古い豆の香り |
| 寂しい | 窓の外を見る、カップを両手で包む | 閉店後のカウンター、最後の客が帰った後 |

## ターミナルUI固有の演出技法

ターミナル（Ratzilla/ratatui）の制約を「味」に変える表現テクニック。

### テキスト表示の「間」

```
通常テキスト:
　佐倉が席についた。いつもの深煎りブレンド。

間を使った演出:
　佐倉が席についた。

　……。

　「いつもの、お願いします」

→ 空白行が「沈黙」「時間の経過」「感情の整理」を表現する
```

### 枠線・罫線の感情的利用

```
通常のシーン区切り:
───────────────────

感情的な転換:
━━━━━━━━━━━━━━━━━━━（太線 = 重い転換）

穏やかな時間経過:
· · · · · · · · · · · · ·

記憶・回想の開始:
┈┈┈┈┈ 3年前 ┈┈┈┈┈
```

### テキストADVの「ページ送り」感覚

ターミナルUIではスクロールではなく「画面切り替え」でテキストを見せる。
これはノベルADVのクリック送りと同じリズムを生む。

- 1画面のテキスト量: 最大10〜15行
- 重要なセリフは1画面に1つだけ表示（「見開き演出」）
- 画面切り替え時に一瞬の空白を入れる（ビジュアルノベルの暗転に相当）

### モノローグの視覚的区別

```
地の文:
　夕方のカフェは静かだった。最後の客が帰って、
　カウンターにはカップが二つ。

モノローグ:
　（この匂いに慣れてきた。いつからだろう）

手紙・レシピノート:
　┌────────────────────────────┐
　│ しずくへ——                    │
　│ このカフェは、いつか誰かが    │
　│ 必要とする時に開けばいい。    │
　└────────────────────────────┘

回想（他キャラの語り）:
　「あの人はね」宮内が目を細めた。
　＞ 時雨さんは、いつもカウンターの端に座って
　＞ 自分が淹れたコーヒーを飲んでいたんだよ。
　＞ 不思議な人でしょう。自分の店で、自分が客みたいに。
```
