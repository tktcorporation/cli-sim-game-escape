# 廃墟カフェ復興記 — キャラクター設定

## 設計思想

全キャラクターに以下の3層構造を持たせる：

1. **表の顔**: 初対面で見える印象。第一印象
2. **裏の顔**: 親密度が上がると見えてくる本当の姿
3. **内面の矛盾**: キャラを立体的にする「自分でも整理できていない感情」

「いい人」「悪い人」は存在しない。全員が何かを抱えていて、
カフェという場所で少しずつそれを見せていく。

### 開示の原則

- **表の顔は「アイコン的わかりやすさ」を担う**: 初見で「こういう人」と掴める特徴
- **裏の顔は「人間的共感」を担う**: 「自分もそういうことあるかも」と思わせる弱さ
- **矛盾は「設定」ではなく「物語」で開示する**: 親密度が上がるにつれ、行動やセリフの端々から滲み出させる
- **段階的開示が継続動機に直結する**: 「もっとこの人のことを知りたい」がプレイの動機になる

## 主人公

### 柊 陽太（ひいらぎ ようた）

| 項目 | 設定 |
|------|------|
| 年齢 | 24歳 |
| 一人称 | 僕 |
| 前職 | 食品メーカーの営業（1年半で退職） |
| 性格表層 | 穏やか、真面目、少し受け身 |
| 性格深層 | 強い意志はあるが、自分の気持ちを言語化するのが苦手 |

**表の顔**: 「なんとなく」カフェを始めた、ちょっと頼りない青年。
会社を辞めた理由も「合わなくて」としか言わない。

**裏の顔**: 実は食品メーカー時代、こだわりの商品を企画したが
「売れないから」と上に潰された。「自分が良いと思ったものを
自分の手で届けたい」という強い願望がある。
ただそれを「夢」と呼ぶのが恥ずかしくて、言えない。

**内面の矛盾**: 「自分の意思でここにいる」と思いたいのに、
実際は「他に行く場所がなかった」面もある。
カフェが上手くいくほど、その矛盾が鮮明になる。

**口調サンプル**:
```
「あ、いらっしゃいませ。……今日は何にしますか」
「いや、まだ全然。手探りですよ、ほんとに」
「……うまく言えないんですけど、この店、嫌いじゃないんです」
```

**成長アーク**:
受け身 → 「自分の店」への当事者意識 → 前オーナーの想いの継承 →
「僕がここにいる理由は、僕が決める」

## 常連客（メインキャスト）

### 佐倉 美月（さくら みつき）

| 項目 | 設定 |
|------|------|
| 年齢 | 32歳 |
| 一人称 | 私 |
| 職業 | フリーランスの校正者（在宅ワーク） |
| 好みのメニュー | 深煎りブレンド（ブラック） |
| 来店パターン | 平日午後、ほぼ毎日 |
| 登場 | Ch.0（最初の客） |

**表の顔**: 物静かで品のある女性。本を読みながらコーヒーを飲む。
多くを語らないが、柊の淹れるコーヒーを「悪くない」と評する。
感情の起伏が少なく、落ち着いた印象。

**裏の顔**: 元パティシエ。かつて自分の小さなケーキ店を持っていたが、
3年前に経営難で畳んだ。カフェ「月灯り」の前オーナー・時雨の
常連だったが、閉店後の2年間は足を運んでいなかった。
柊がカフェを再開したと聞いて「どんな人がやっているのか」確認しに来た。

**内面の矛盾**: 「もう飲食業には関わらない」と決めたはずなのに、
このカフェに通い続けている自分が怖い。柊がメニュー開発で
悩んでいるのを見ると口を出したくなるが、それは
「また自分が傷つくかもしれない世界に足を踏み入れること」でもある。

**口調サンプル**:
```
「……少し苦いですね。でも、嫌いじゃないです」
「聞きたいことがあるなら聞けばいいのに。回りくどい人ですね」
「——私に訊かないでください。もう、そっち側の人間じゃないので」
「……ごめんなさい。今のは、忘れてください」
```

**キャラエピソードの方向性**:
「もう一度作る側に立つ」ことへの恐怖と憧れ。
柊の不器用だが真摯な姿勢に少しずつ心を開き、
最終的にカフェのスイーツメニュー開発を手伝うようになる。
ただしそれは「柊のため」であって「自分のため」ではない、
と自分に言い聞かせている。

### 天野 蓮（あまの れん）

| 項目 | 設定 |
|------|------|
| 年齢 | 19歳 |
| 一人称 | 俺 |
| 職業 | 大学1年生（経営学部）、商店街でバイト中 |
| 好みのメニュー | 甘い系（キャラメルラテ、ココア） |
| 来店パターン | 夕方〜夜、週3〜4回 |
| 登場 | Ch.1 |

**表の顔**: 明るくて人懐っこい大学生。誰とでもすぐ打ち解ける。
商店街の顔の広いバイトくんで、情報源としても頼りになる。
将来は「起業する」と公言している。

**裏の顔**: 実家は商店街で30年続いた金物屋だったが、去年廃業した。
「起業する」は本心だが、その動機の半分は
「親の店を救えなかった」負い目の裏返し。
経営学部に入ったのも「次は失敗しないため」。

**内面の矛盾**: 商店街の個人店を応援したい気持ちは本物。
でも同時に「個人店は大型店に勝てない」と
心のどこかで思っている。柊のカフェを手伝いながら、
その矛盾に気づいていく。

**口調サンプル**:
```
「柊さん、ここの内装もうちょい何とかなりません？」
「俺、将来は絶対自分の店やるんすよ。いや、店っていうか会社？」
「——うちの親父も、最後まで諦めなかったんすけどね」
「……いいっす、なんでもないです。ココアおかわりいいすか」
```

**キャラエピソードの方向性**:
「成功」とは何か。大きくすることなのか、続けることなのか。
柊と商店街の人々を通じて、父親の店が「失敗」ではなかったことに
気づいていく物語。

### 神崎 凛（かんざき りん）

| 項目 | 設定 |
|------|------|
| 年齢 | 28歳 |
| 一人称 | あたし |
| 職業 | 地方紙の記者 |
| 好みのメニュー | その日のおすすめ（新しいもの好き） |
| 来店パターン | 不定期、取材の合間 |
| 登場 | Ch.2 |

**表の顔**: サバサバしていて行動力がある。「面白い」が行動基準。
廃墟カフェの復興を記事にしようと取材に来たのがきっかけ。
社交的で会話が上手い。

**裏の顔**: 元々は東京の大手出版社勤務だったが、自分の書いた記事が
炎上したことがトラウマで地方に転職した。
「面白いことを発信したい」欲求と、「また傷つくかもしれない」恐怖の
間で揺れている。地方紙の仕事は安全だが、物足りない。

**内面の矛盾**: 人の本音を引き出すのは得意だが、自分の本音は出さない。
取材者としての「外側から見る」立場が心地よいが、
それは同時に「当事者にならない」逃げでもある。

**口調サンプル**:
```
「あ、撮っていい？ 記事にするかも。しないかも」
「へえ、面白いじゃん。その話、もう少し聞かせてよ」
「——あたしはさ、書く側だから。作る側にはなれないの」
「……あの記事、読まないでね。昔の、恥ずかしいやつだから」
```

**キャラエピソードの方向性**:
「伝える」ことの意味を再発見する。商店街の人々の物語を
取材するうちに、自分も「外側」ではいられなくなる。
カフェの復興ドキュメンタリーを書くことが、
彼女自身の再起にもなっていく。

### 宮内 孝之（みやうち たかゆき）

| 項目 | 設定 |
|------|------|
| 年齢 | 58歳 |
| 一人称 | 私（ゆっくり、丁寧な口調） |
| 職業 | 古書店「栞堂」店主 |
| 好みのメニュー | ほうじ茶、和菓子に合うもの |
| 来店パターン | 週末午前、ゆっくり |
| 登場 | Ch.1 |

**表の顔**: 穏やかで物知りな老紳士。商店街の長老的存在。
前オーナー・時雨の親友だった。柊に商店街の歴史や
時雨のことを少しずつ教えてくれる、メンター的な人物。

**裏の顔**: 時雨が消えた理由を唯一知っている（と匂わされる）人物。
しかし、それを柊に伝えるべきかどうか迷っている。
時雨の娘のことも知っているが、
「自分が口を出すことではない」と線を引いている。

**内面の矛盾**: 柊のことを応援したい。でも、時雨と同じ道を
辿るのではないかと心配もしている。助言したいが、
助言が「干渉」になることを恐れている。
「若い人のやることに口を出さない」が信条だが、
本当はもっと手を差し伸べたい。

**口調サンプル**:
```
「やあ、今日も良い香りだ。この角を曲がると、もう分かるよ」
「時雨さんはね……不思議な人でしたよ。いつも笑っていた」
「——その箱の中身は、君が自分で見なさい。私からは、言えない」
「老人の心配だと思って、聞き流してくれていいんだが……」
```

**キャラエピソードの方向性**:
「見守る」ことの難しさと美しさ。メインストーリーの鍵を握る人物であり、
柊が真相に近づく過程で「いつ、何を、どこまで話すか」を
常に判断し続ける。最終的に柊に時雨の手紙を渡す。

### 桐谷 楓（きりたに かえで）

| 項目 | 設定 |
|------|------|
| 年齢 | 26歳 |
| 一人称 | 私 |
| 職業 | 「Café BLANC」エリアマネージャー |
| 好みのメニュー | エスプレッソ系（プロの目で品定めしている） |
| 来店パターン | Ch.3以降、不定期 |
| 登場 | Ch.3（ライバル登場） |

**表の顔**: 洗練された雰囲気のキャリアウーマン。
チェーンカフェ「Café BLANC」のエリア展開責任者として
この商店街エリアに出店計画を進めている。
柊に対しては丁寧だが、どこか上から目線。

**裏の顔**: カフェ業界で結果を出し続けてきたが、
「効率」と「味」の間で常に葛藤している。
チェーンの本部からは数字を求められるが、
本当は「この1杯」にこだわりたい人間。
柊のカフェの非効率さに苛立ちつつも、
その「贅沢さ」が羨ましい。

**内面の矛盾**: 柊のやり方を「甘い」と思っている自分と、
「あれが正しいのかもしれない」と思う自分がいる。
仕事で成功することと、自分が信じることの間で引き裂かれている。

**口調サンプル**:
```
「素敵なお店ですね。——商売としては、少し心配ですけど」
「数字は嘘をつかないの。情熱だけじゃ、家賃は払えないでしょう？」
「——ねえ、なんでそんなに楽しそうなの。赤字なのに」
「……私だって、最初はこういう店がやりたかった」
```

**キャラエピソードの方向性**:
敵対→理解→共存。最初は「商店街から個人店を追い出す」存在に見えるが、
彼女自身もシステムの中で苦しんでいることが分かる。
最終的に「Café BLANCのメニューに月灯りの限定ブレンドを」
という提案をする — それが本部への反抗でもあり、共存の証でもある。

## ストーリーNPC（メインストーリー関連）

### 藤宮 時雨（ふじみや しぐれ）— 前オーナー

| 項目 | 設定 |
|------|------|
| 年齢 | 60歳（失踪時） |
| 登場形式 | 回想、手紙、レシピノート、他キャラの証言 |
| 直接登場 | なし（最終章の手紙のみ） |

**人物像**: 穏やかで少し不思議な老人。
コーヒーの腕は確かだが、経営には無頓着だった。
「おいしいコーヒーがあれば、人は来る」が口癖。
商店街の誰からも慕われていたが、不器用な父親だった。

**メインストーリーでの役割**: 直接登場しないからこそ、
他キャラの語り・遺したもの（レシピノート、手紙、カフェそのもの）を通じて
「不在の存在感」を描く。プレイヤーはゲームを進めるうちに、
会ったことのない時雨に愛着を持つようになる——それが狙い。

### 藤宮 雫（ふじみや しずく）— 時雨の娘

| 項目 | 設定 |
|------|------|
| 年齢 | 30歳 |
| 職業 | グラフィックデザイナー（東京在住） |
| 登場 | Ch.6以降（電話→Ch.8で来店） |

**人物像**: 父（時雨）のカフェを継がず、デザイナーの道を選んだ。
そのことで父と10年以上疎遠になっている。
父を愛しているが、「カフェを継がなかった自分」を
ずっと後ろめたく思っている。

**内面の矛盾**: 自分の夢を選んだことは正しかったと思う。
でも、父の店が廃墟になっている現実を見ると、
自分が捨てたものの大きさに押しつぶされそうになる。
