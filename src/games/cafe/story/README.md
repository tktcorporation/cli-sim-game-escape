# 廃墟カフェ復興記 — シナリオバイブル

## ファイル構成

| ファイル | 内容 | 用途 |
|---------|------|------|
| [WORLD.md](./WORLD.md) | 世界観・舞台設定 | 場所、時代、カフェの詳細、商店街の設定 |
| [CHARACTERS.md](./CHARACTERS.md) | キャラクター設定 | 全キャラの3層構造（表・裏・矛盾）、口調、関係図 |
| [STYLE_GUIDE.md](./STYLE_GUIDE.md) | 文体ガイド | 視点、文体ルール、禁止事項、感情表現ストック |
| [PLOT.md](./PLOT.md) | プロット構成 | チャプター構成、感情設計、伏線管理、サイドクエスト |
| [../DESIGN.md](../DESIGN.md) | ゲームデザイン | システム設計、ソシャゲ運用基盤、実装フェーズ |

## 使い方

### シナリオを書くとき

1. **STYLE_GUIDE.md** を読んで文体ルールを確認
2. **CHARACTERS.md** で該当キャラの口調・内面を確認
3. **PLOT.md** でそのチャプターの感情設計・伏線を確認
4. **WORLD.md** で舞台の空気感を確認

### 新キャラを追加するとき

1. **CHARACTERS.md** の設計チェックリストに沿って設定
2. **PLOT.md** のチャプター構成に登場箇所を追加
3. **WORLD.md** に関連する場所があれば追加

### 新チャプター/サイドクエストを追加するとき

1. **PLOT.md** の感情設計表を確認し、感情の流れに矛盾がないか確認
2. 伏線管理表を更新
3. 解放条件をDESIGN.mdのシステムと整合させる
